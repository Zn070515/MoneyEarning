# 用户版授权指南

> 版本 v1.0 | 2026-05-24 | 面向最终用户

## 一、授权概述

QuantVault 采用**离线激活授权**模式：

1. 安装后自动获赠 **14 天全功能试用**
2. 试用满意后购买，获得**永久激活码**
3. 激活码**绑定一台电脑**（按硬件指纹）
4. 无需联网验证，无年费/月费

---

## 二、试用期

### 如何开始试用

安装 QuantVault 并首次启动，自动开始 14 天全功能试用。

试用期内：所有付费功能均可用。

### 如何查看试用剩余天数

启动软件后，标题栏显示"试用版 - X 天剩余"。

或：帮助 → 关于 → 查看授权信息。

### 试用结束后

- 付费功能自动锁定（回测模板限制、扫描器禁用等）
- 免费版功能（K线、指标、数据导入、交易记录等）继续永久可用
- 随时可以购买激活码解锁

## 三、如何购买

### 购买步骤

1. **试用确认**：先用 14 天，确认软件满足需求
2. **联系购买**：通过官网联系方式
3. **付款**：微信/支付宝 ¥199（一次买断，永久有效）
4. **提供机器码**：帮助 → 关于 → 查看机器码（格式 `XXXX-XXXX-XXXX-XXXX`）
5. **获取激活码**：我们收到机器码后生成并发送给您
6. **输入激活**：帮助 → 输入激活码 → 粘贴 → 点击激活

### 价格

- **¥199 / 终身**（一次性买断，不需续费）
- 同大版本内免费更新（如 v1.0 → v1.x）
- 重大版本升级（如 v1.x → v2.0）可能另收升级费，但已购用户享优惠价

## 四、激活操作

### 激活步骤

1. 打开 QuantVault
2. 点击菜单：**帮助 → 输入激活码**
3. 粘贴您收到的激活码（格式 `QV-XXXX-XXXX-XXXX-XXXX`）
4. 点击"激活"
5. 提示"激活成功"，所有付费功能立即解锁

### 激活不成功怎么办

参见本文档第七节"激活失败怎么办"。

## 五、更换电脑 / 重装系统

您的激活码绑定在购买时的电脑硬件上。如果更换电脑或重装系统，硬件指纹可能变化，激活码会失效。

### 处理流程

1. 在新电脑 / 重装后的系统安装 QuantVault
2. 打开软件 → 帮助 → 关于 → 获取**新机器码**
3. 联系我们，提供以下信息：
   - **旧机器码**（或原激活码）
   - **新机器码**
4. 我们验证后免费签发新激活码

### 限制

- 每用户每半年可免费更换 1 次
- 超过频率需要人工审核
- 如发现恶意换绑（明显不同硬件频繁切换），将停止售后

### 重装前建议

重装系统前：
1. 备份数据：复制 `C:\Users\你的用户名\AppData\Roaming\com.quantvault.app\` 整个目录
2. 记录激活码：记下你当前的激活码
3. 重装后恢复数据到相同位置，再联系我们换绑

## 六、备份与恢复数据

独立的授权，你的分析数据和授权状态都存储在本地。

### 备份

打开 QuantVault → 设置 → 数据管理 → "备份数据" → 选择保存位置。

### 恢复

打开 QuantVault → 设置 → 数据管理 → "恢复数据" → 选择备份文件。

注意：恢复会自动备份当前数据，不会造成数据丢失。

## 七、激活失败怎么办

| 提示 | 原因 | 解决 |
|------|------|------|
| "激活码格式错误" | 输入有误 | 检查是否完整复制，格式为 QV-XXXX-XXXX-XXXX-XXXX |
| "激活码无效" | 签名验证失败 | 联系重新签发 |
| "机器码不匹配" | 此激活码绑定了其他电脑 | 提供新机器码申请换绑 |
| "激活码已过期" | — | 激活码不会过期，此错误不应出现，联系排查 |
| "无法获取机器指纹" | 权限不足 | 尝试以管理员身份运行 |
| "数据库写入失败" | 磁盘空间不足或权限问题 | 检查磁盘空间，确认数据目录可写 |

## 八、常见问题

### Q: 激活码能用在多台电脑吗？
A: 不能。一个激活码绑定一台电脑。如需多台使用，需购买多个激活码。

### Q: 激活码有有效期吗？
A: 没有。激活后永久有效。

### Q: 需要联网验证吗？
A: 不需要。激活验证在软件内本地完成。

### Q: 换了 CPU / 内存 / 硬盘，激活会失效吗？
A: 单独更换内存/硬盘一般不影响。更换 CPU/主板可能导致机器码变化。如失效，联系我们换绑。

### Q: 虚拟机里能用吗？
A: 未测试也不推荐。虚拟机环境下的硬件指纹不稳定，可能频繁失效。

### Q: 买了之后可以退款吗？
A: 激活码一经发送，原则上不支持退款。详见《退款政策》。建议先用 14 天试用确认。
