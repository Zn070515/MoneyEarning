# 激活失败情况（20+ 种）

> 版本 v1.0 | 2026-05-24

---

## 一、激活码格式问题

### 1. 激活码为空

**现象**：点击激活时输入框为空
**提示**："请输入激活码"
**处理**：提示用户粘贴激活码

### 2. 激活码格式错误（长度不对）

**现象**：输入框内容长度不等于 31
**提示**："激活码格式错误。正确格式：QV-XXXX-XXXX-XXXX-XXXX"
**处理**：检查是否完整复制，去除前后空格

### 3. 激活码含非法字符

**现象**：Base32 解码失败
**提示**："激活码包含无效字符，请检查是否完整复制"
**处理**：重新发送原激活码，让用户核对

### 4. 分隔符错误

**现象**：分隔符不是 `-`
**提示**："激活码格式错误"
**处理**：自动规范化（去空格、统一分隔符），再验证

### 5. 大小写问题

**现象**：用户输入全小写
**处理**：系统自动转换为大写，大小写不敏感

## 二、签名验证失败

### 6. RSA 签名不匹配

**现象**：公钥验签返回 false
**提示**："激活码验证失败。如确认已完整复制激活码，请重新获取。"
**原因**：
- 激活码被篡改
- 激活码复制不完整
- 私钥/公钥不匹配（极罕见）
**处理**：重新生成激活码

### 7. 校验段 CRC32 不匹配

**现象**：CRC32 检查失败
**提示**："激活码验证失败"
**原因**：激活码某段输入错误
**处理**：让用户逐段核对，或重新发送

---

## 三、机器码问题

### 8. 机器码不匹配

**现象**：激活码中的机器码哈希 ≠ 当前机器码哈希
**提示**："此激活码绑定的设备与当前设备不匹配。如已更换电脑，请联系获取新激活码。"
**原因**：
- 换了电脑
- 换了 CPU/主板
- 虚拟机/云桌面环境
**处理**：走换绑流程

### 9. 无法获取机器指纹

**现象**：读取 WMI 信息失败
**提示**："无法获取设备信息，请检查系统权限或联系支持。"
**原因**：
- WMI 服务被禁用
- 权限不足
- 特殊系统环境
**处理**：
- 尝试管理员身份运行
- 检查 WMI 服务：`services.msc` → Windows Management Instrumentation → 启动

### 10. 机器码格式变化

**现象**：同一台电脑，机器码偶尔变化
**提示**：软件启动时检测到机器码变化（vs 上次记录的）
**原因**：
- BIOS 更新
- 系统重大更新
- 硬件驱动更新
**处理**：走换绑流程

### 11. 虚拟化环境机器码不稳定

**现象**：每次重启机器码都不同
**提示**："检测到设备环境不稳定，授权可能频繁失效。"
**原因**：虚拟机/云桌面环境中硬件指纹不稳定
**处理**：建议在物理机上使用

## 四、数据库问题

### 12. 授权数据库不存在

**现象**：`licenses` 表查询时数据库文件不存在
**提示**："数据文件异常，请重新安装软件。"
**原因**：数据库文件被删除或损坏
**处理**：重新安装，自动重建数据库

### 13. 授权记录写入失败

**现象**：INSERT 到 licenses 表失败
**提示**："激活信息保存失败，请检查磁盘空间和写入权限。"
**原因**：
- 磁盘空间不足
- 文件权限问题（如安装在需要管理员权限的位置）
- 数据库被其他进程锁定（WAL 模式下降概率低）

### 14. 数据库文件只读

**现象**：SQLite 返回 SQLITE_READONLY
**提示**："无法写入授权信息，请检查数据目录权限。"
**原因**：
- 数据目录权限为只读
- 磁盘被写保护
- 杀软拦截写入
**处理**：检查目录权限，添加杀软信任

## 五、软件版本问题

### 15. 旧版软件不支持激活

**现象**：v0.9.x 及更早版本无激活功能
**提示**："当前版本不支持激活功能，请升级到最新版本。"
**处理**：重新下载最新版本安装

### 16. 新版授权格式变化

**现象**：旧激活码格式在新版本中不兼容
**处理**：重新生成新格式激活码

## 六、试用与付费状态冲突

### 17. 试用期内激活

**现象**：试用尚未结束就激活
**提示**："激活成功，您的试用期已转为永久授权。"
**处理**：正常流程，更新 `license_type` 从 trial → pro

### 18. 已激活再次输入激活码

**现象**：pro 用户再次输入（其他）激活码
**提示**："当前已是永久授权，无需再次激活。"
**处理**：不需要操作，忽略

### 19. 试用过期后激活

**现象**：试用已过期，付费功能锁定，用户才激活
**提示**："激活成功，付费功能已恢复。"
**处理**：正常流程

---

## 七、人为问题

### 20. 一个激活码多人使用

**现象**：同一个激活码在不同机器上尝试激活
**结果**：只有与激活时绑定的机器码一致的那台能通过
**提示**（其他人）："激活码与设备不匹配"
**处理**：告知一码一机政策

### 21. 用户使用破解/盗版激活码

**现象**：激活码格式正确但签名验证失败
**提示**："激活码无效"
**处理**：不予处理，建议从官方渠道购买

### 22. 激活码分享到网上

**现象**：同一个激活码被多人咨询
**处理**：
1. 调查使用情况
2. 如确认泄露，联系原购买用户
3. 视情况停用该激活码（换绑到新机器码并提醒用户）
4. 新激活码不外泄

## 八、开发者侧问题

### 23. 私钥丢失

**现象**：无法为新机器码生成激活码
**影响**：之前所有发出的激活码正常使用（公钥验证只读），但无法签发新码或换绑
**处理**：
1. 生成新的密钥对
2. 更新软件中的公钥
3. 发布新版本
4. 为所有现有付费用户重新签发激活码（用旧密钥验证用户身份）
**预防**：私钥多份离线加密备份

### 24. 生成工具编译失败

**现象**：`cargo build` 失败
**处理**：检查 Rust 版本、依赖更新、Cargo.toml

### 25. 生成激活码时机器码格式错误

**现象**：用户提供的机器码格式不标准
**处理**：自动规范化（去分隔符、去空格、统一大写）

## 九、错误码体系

### 建议实现的错误码

| 错误码 | 含义 | 用户提示 |
|:---:|------|------|
| L001 | 激活码为空 | 请输入激活码 |
| L002 | 激活码格式错误 | 激活码格式错误，正确格式：QV-XXXX-XXXX-XXXX-XXXX |
| L003 | 激活码含非法字符 | 激活码包含无效字符 |
| L004 | CRC32 校验失败 | 激活码验证失败，请检查是否完整复制 |
| L005 | RSA 签名验证失败 | 激活码无效 |
| L006 | 机器码不匹配 | 此激活码绑定了其他设备 |
| L007 | 机器指纹获取失败 | 无法获取设备信息 |
| L008 | 数据库写入失败 | 激活信息保存失败 |
| L009 | 数据库读取失败 | 无法读取授权信息 |
| L010 | 版本不支持 | 当前版本不支持此功能 |
| L011 | 已是永久授权 | 当前已是永久授权 |
| L012 | 试用尚未开始 | 授权状态异常 |

---

> **原则**：每次激活失败都必须有明确的原因说明和建议操作。不能只显示"激活失败"四个字。
