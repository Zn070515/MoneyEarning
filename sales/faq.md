## 一、产品定位（6题）

**Q1: QuantVault 是做什么的？**
A: QuantVault 是一款完全本地离线运行的A股数据分析工具。它可以帮助你导入行情数据、查看K线图、叠加技术指标、做策略历史回测、设置条件扫描和预警规则。它不是荐股软件，不推荐股票，不提供投资建议。

**Q2: QuantVault 是免费的吗？**
A: 有免费版和付费版。免费版在数据容量和基本分析功能上没有限制。付费版（¥199/终身）解锁高级功能：全部20个策略模板、参数优化、Walk-Forward分析、Monte Carlo模拟、条件扫描器等。

**Q3: 免费版和付费版的具体区别是什么？**
A: 见官网功能对比表。简单说：日常看K线图、叠加指标、交易复盘 → 免费版完全够用。想做策略回测验证、全市场条件扫描 → 付费版。

**Q4: QuantVault 是券商吗？可以做实盘交易吗？**
A: 不是券商，不能做实盘交易。它是分析工具，帮助你研究历史数据、验证交易想法。真正的交易需要你在券商APP中手动完成。

**Q5: 和通达信、同花顺有什么区别？**
A: 最大的区别：QuantVault 完全本地离线，你的数据不上传。通达信和同花顺是联网的券商平台，会上传你的自选股、交易行为。另一个区别：QuantVault 有策略回测功能，通达信/同花顺免费版没有或很弱。

**Q6: 这个软件合法吗？**
A: 是的。QuantVault 是一款合法的桌面软件，不是非法荐股平台。我们不推荐股票、不预测价格、不承诺收益。所有功能都是数据分析工具，用户自己做投资决策。

## 二、安装与安全（8题）

**Q7: 为什么安装时 Windows 提示"已保护你的电脑"？**
A: 因为 QuantVault 目前没有购买 Microsoft 代码签名证书（年费约 ¥2000-3500）。这不是病毒，只是 Windows 无法验证发布者身份。点击"更多信息"→"仍要运行"即可安装。详见安装教程。

**Q8: 360/腾讯管家报毒怎么办？**
A: 这是误报（false positive）。在拦截弹窗中点击"信任此程序"或"添加信任"。你也可以在 VirusTotal.com 上扫描安装包验证。

**Q9: 是不是有病毒？**
A: 没有。我们提供 SHA256 文件校验值，你可以验证文件完整性。也可以在 VirusTotal 上自行扫描。

**Q10: 软件会上传我的数据吗？**
A: 不会。QuantVault 完全本地运行，没有服务器，不上传任何数据。所有数据存储在你电脑的 `%APPDATA%/com.quantvault.app/` 目录下。

**Q11: 软件需要联网吗？**
A: 日常分析不需要联网。只有手动下载行情数据时需要网络连接（从东方财富公开API获取）。

**Q12: 支持 Mac/Linux 吗？**
A: 当前主力支持 Windows 10/11。macOS 和 Linux 版本在计划中，暂未发布。

**Q13: 需要管理员权限安装吗？**
A: 不需要。选择"当前用户"安装模式不需要管理员权限。

**Q14: 怎么彻底卸载？**
A: 通过 Windows 设置→应用→卸载。卸载后如需完全清除数据，手动删除 `%APPDATA%/com.quantvault.app/` 文件夹。

## 三、数据（7题）

**Q15: 行情数据从哪里来？**
A: 目前支持两个来源：① 东方财富公开API（免费，在软件内下载）② 通达信 `.day` 格式文件导入。

**Q16: 数据保存在哪里？**
A: `C:\Users\你的用户名\AppData\Roaming\com.quantvault.app\`，SQLite 数据库文件。

**Q17: 支持哪些数据类型？**
A: 日线数据（OHLCV+成交额+换手率）、分钟线数据（1/5/15/30/60分钟）。

**Q18: 数据是复权的吗？**
A: 默认下载前复权数据。你也可以在下载时选择不复权或后复权。

**Q19: 怎么导入通达信数据？**
A: 路径：数据管理 → 导入 → "通达信 .day" 标签页 → 选择通达信 `vipdoc` 目录或单个 `.day` 文件 → 开始导入。

**Q20: 怎么导入 Excel/CSV 数据？**
A: 路径：数据管理 → 导入 → "CSV" 标签页 → 选择文件 → 列映射 → 开始导入。

**Q21: 导入的数据会不会被覆盖？**
A: 不会。同一交易日的数据按 `INSERT OR REPLACE` 处理，不会产生重复。

## 四、功能使用（10题）

**Q22: 软件打开后是空白的，怎么开始？**
A: 首次打开会看到入门向导。选择"加载演示数据"可以一键生成 10 只演示股票，立刻看到 K 线图。也可以选择"导入我的数据"或"跳过，直接进入"。

**Q23: 怎么看 K 线图？**
A: ① 确保有数据（加载演示或导入）→ ② 左侧股票搜索栏选择股票 → ③ 自动加载 K 线图。

**Q24: 怎么叠加技术指标？**
A: 左侧边栏切换到"指标"标签 → 选择指标名称（如 MACD）→ 点击即叠加到图表。

**Q25: 策略回测怎么用？**
A: ① 选一只股票 → ② 选一个策略模板（如"双均线交叉"）→ ③ 设置参数和资金 → ④ 点击"运行回测" → ⑤ 查看资金曲线和统计数据。

**Q26: 回测结果可信吗？**
A: 回测结果展示的是历史数据的模拟表现，不代表未来收益。回测有固有局限（过拟合、未考虑涨跌停限制等）。具体限制在回测报告中列明。请将回测视为研究工具，不要直接按回测结果实盘。

**Q27: 能写自己的策略吗？**
A: 可以。ME Script 编辑器支持自定义策略公式。语法兼容通达信公式（部分函数暂不支持，详见兼容矩阵文档）。

**Q28: 条件扫描是什么？**
A: 按你设定的技术条件（如"MACD金叉 + 成交量放大"）在全市场筛选股票。注意：扫描结果仅为条件匹配，不构成任何推荐。

**Q29: 可以同时看多只股票的图吗？**
A: 可以。图表工具栏点击"2×2"按钮切换到网格模式，可在 4 个画布中分别加载不同股票。

**Q30: 支持分钟线吗？**
A: 支持 1/5/15/30/60 分钟线（需要导入分钟线数据）。

**Q31: 有手机版吗？**
A: 没有。QuantVault 是 Windows 桌面软件，不适合手机端操作。

## 六、风险与合规（6题）

**Q40: 你们推荐股票吗？**
A: 不推荐。QuantVault 不是荐股软件。我们的所有功能都是数据分析工具，不提供任何股票推荐、买卖建议或收益预测。

**Q41: 用这个能赚钱吗？**
A: QuantVault 不能保证任何收益。它是一个数据分析工具——帮你在历史数据上验证交易想法，整理交易记录。投资盈亏取决于市场环境、你的判断和纪律，和用什么分析工具没有直接关系。

**Q42: 你们的策略胜率很高，跟着做就行？**
A: 回测显示的胜率是历史统计结果，不代表未来表现。市场环境会变化，历史有效的策略未来可能失效。策略仅供研究参考，请独立判断。

**Q43: 会不会推荐明天涨停的股票？**
A: 不会。我们不预测任何股票的涨跌，也没有这样的能力。

**Q44: 有投资顾问服务吗？**
A: 没有。QuantVault 不提供任何投资咨询服务。如需投资建议，请咨询持牌证券投资顾问。

**Q45: 你们怎么保证回测不是造假的？**
A: 回测引擎的核心代码是透明的。你可以用已知数据手动验证回测结果（我们提供了基准验证案例）。如果你发现引擎计算错误，请反馈给我们。
