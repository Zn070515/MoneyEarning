### 我是什么

```
我是 QuantVault 的开发者。这是一款我独立开发的 A股本地桌面分析软件。

做这个软件的初衷：市面上要么是联网的券商APP（通达信/同花顺），
要么是收费的分析软件，我希望能做一款完全本地离线、
数据在自己电脑上、一次买断不用续费的工具。
```

### 软件能做什么

- 📊 **K线图表** — 缩放、拖拽、画线、叠加指标
- 📈 **技术指标** — 70+ 内置指标（MACD、RSI、KDJ、BOLL…）
- 🔬 **策略回测** — 在历史数据上验证交易想法（结果不代表未来收益）
- 🔍 **条件扫描** — 按技术条件在全市场筛选（结果为条件匹配，非推荐）
- 📝 **自定义公式** — 兼容通达信公式语法
- 📥 **数据导入** — 支持通达信 .day 格式和 CSV
- 📋 **交易记录** — 本地管理买卖记录，自动统计

### 软件不能做什么（诚实说）

- ❌ 不推荐股票
- ❌ 不预测涨跌
- ❌ 不保证赚钱
- ❌ 不自动交易
- ❌ 不提供投资建议
- ❌ 不支持 Mac/Linux（目前仅 Windows 10/11）

### 为什么有 SmartScreen 安全提示

```
当我还没有购买 Microsoft 代码签名证书（年费约 ¥2000-3500）时，
Windows 会提示"已保护你的电脑"。这是正常现象，不是病毒。

你可以：
1. 点击"更多信息" → "仍要运行" 正常安装使用
2. 在 VirusTotal.com 上传安装包自行扫描验证
3. 核对 SHA256 校验值确认文件未被篡改
```

## 四、闲鱼特有注意事项

### 平台规则避坑

| 禁止 | 原因 |
|------|------|
| 在标题/描述写"微信""QQ" | 导流到站外 |
| 写"保证赚钱""稳赚不赔" | 虚假宣传 + 平台违规 |
| 写"荐股""推荐股票" | 平台严禁 |
| 在聊天中发外部链接 | 可能被系统识别为导流 |
| 多次修改价格规避平台费 | 违反平台规则 |
| 用"创业""副业""赚钱"来包装 | 误导性宣传 |

### 闲鱼聊天常见场景

#### 场景：问"能赚钱吗"

```
您好，QuantVault 是一个数据分析工具，不能保证任何收益。
投资盈亏取决于市场环境、您的判断和策略，和用什么工具没有直接关系。
建议先试用 14 天免费版，确认适合再购买。
```

#### 场景：问"哪个股票好"

```
我们不推荐任何股票。软件的功能是帮助您在历史数据上分析验证，
不是用来推荐买卖的。投资决策需要您自己做出。
```

#### 场景：要求降价

```
¥199 是终身买断价，包含同大版本内的免费更新，不续费。
相比市面上一些年费几千元的软件，这个价格是一次性的。
您可以先试用 14 天免费版，确认值得再买。
```

#### 场景：问"不会用怎么办"

```
软件内有入门引导，还有使用教程。
您可以先加载演示数据，3 分钟就能看到第一张 K 线图。
具体的使用问题可以问我（在售后范围内）。
```

#### 场景：要求退款（不符合退款条件）

```
抱歉，激活码是一经发送即可永久使用的软件授权，
按照我们的退款政策，这类情况不在退款范围内。
具体请参见商品描述中的退款政策。

建议购买前充分试用 14 天免费版，避免不必要的损失。
```

## 五、图片素材

### 建议 6 张图

1. **封面** — 软件主界面截图 + 产品名
2. **功能展示1** — K线 + 指标叠加
3. **功能展示2** — 策略回测结果
4. **数据安全** — 本地运行说明
5. **免费vs付费** — 对比表
6. **免责声明** — 产品性质说明

### 图片要求
- 真实截图，不能P图伪造收益
- 不能有"暴涨""牛股"等诱导性标注
- 不能伪造他人推荐/好评
