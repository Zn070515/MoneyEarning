## 二、禁止标题词

以下词汇**不得**出现在标题中：
- 荐股、推荐、牛股、黑马、金股
- 赚钱、盈利、稳赚、暴利、翻倍
- 预测、精准、准确率、成功率
- 买卖点、买入信号、卖出信号、抄底、逃顶
- 主力、机构、私募、内幕、资金流向
- 自动交易、跟单、跟投、喊单
- 必涨、涨停、暴涨、拉升、起飞

## 四、商品详情页结构

### 1. 产品定位声明（第一屏）

```
QuantVault 是一款完全本地离线的 A股数据分析桌面工具。

⚠️ 重要声明：
本软件不提供任何证券投资咨询服务，不构成任何买入、卖出或持有建议。
所有功能仅用于历史数据分析和用户自我研究。
历史回测表现不代表未来收益。
投资有风险，入市需谨慎。
```

### 2. 适合谁用

- 习惯用通达信看盘，想找一款本地离线替代工具的投资者
- 有自己交易想法，需要在历史数据上验证策略的研究型投资者
- 需要整理交易记录、复盘盈亏的纪律型交易者
- 想学习技术指标原理的新手投资者

### 3. 不适合谁用

- 期望找到"牛股推荐"的
- 期望"跟单赚钱"的
- 期望"自动交易"的
- 期望"保证收益"的
- 不想花时间学习使用的

### 4. 核心功能（8个功能块）

| 功能 | 说明 |
|------|------|
| K线图表 | 自研 Canvas 引擎，缩放、拖拽、十字光标，支持画线工具 |
| 技术指标 | 70+ 内置指标（MACD、RSI、KDJ、BOLL…），可叠加对比 |
| 策略历史回测 | 20 个策略模板，查看资金曲线、胜率、盈亏比等统计 |
| 条件扫描 | 自定义技术条件，在全市场筛选匹配标的 |
| 自定义公式 | ME Script 兼容通达信公式语法，编写自定义指标 |
| 数据导入 | 支持通达信 .day 格式、CSV 文件导入 |
| 交易记录 | 本地记录买卖操作，自动统计盈亏 |
| 条件提醒 | 本地价格/均线/成交量条件触发提醒 |
