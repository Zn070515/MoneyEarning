# 售后政策

> 版本 v1.0 | 2026-05-24

---

## 一、售后范围

QuantVault 提供有限售后支持，范围包括：

### 我们提供的支持

| 类别 | 内容 |
|------|------|
| **激活问题** | 激活码无法使用、机器码变化、换电脑重签等 |
| **安装问题** | 安装失败、启动闪退、杀软误报处理 |
| **使用问题** | 功能如何操作、数据如何导入、图表如何使用 |
| **Bug 报告** | 软件崩溃、计算错误、界面异常、数据问题 |
| **功能建议** | 收集用户反馈的功能需求 |

### 我们不提供的"支持"

| 类别 | 说明 |
|------|------|
| **投资建议** | 不分析股票、不推荐买卖时机、不评价用户策略 |
| **策略调优** | 不帮你调参数、不告诉你"哪个策略更好" |
| **选股指导** | 不回答"哪只股票好""什么时候买""要不要卖" |
| **代写公式** | 不提供定制 ME Script 编写服务 |
| **1对1教学** | 不提供一对一视频/电话教学 |
| **市场分析** | 不分析大盘走势、不解读政策影响 |

## 二、响应时效

| 渠道 | 响应时间 |
|------|:------:|
| 邮件 | 1-3 个工作日 |
| GitHub Issue | 1-5 个工作日 |
| 微信/QQ（如有） | 工作时间 2 小时内 |

---

## 三、用户行为规范

以下行为可能导致售后支持终止：

- 辱骂、威胁客服人员
- 将激活码分享给他人或转售
- 使用破解/盗版激活码要求售后
- 反复提出已明确不在支持范围内的要求（如投资建议）
- 恶意差评、虚假投诉

---

## 四、版本更新与维护

- **Bug 修复**：免费提供，不限制版本
- **功能更新**：同大版本内免费更新
- **重大版本升级**（如 v1.x → v2.0）：可能另行收费，但对已购买用户提供优惠升级价
- **安全补丁**：始终免费

---

## 五、限制与免责

1. 我们不承诺 7×24 小时支持。QuantVault 是独立开发者项目，非企业级服务。
2. 对于用户因使用或无法使用本软件而产生的任何投资损失，我们不承担责任。
3. 对于用户自行修改软件文件、数据库、注册表而导致的问题，我们不承担责任。
4. 我们保留根据实际情况调整售后政策的权利，调整前将通知用户。

> **原则**：诚实、有限的售后承诺，好过夸张但无法兑现的服务承诺。
