# 回测报告免责声明

> 版本 v1.0 | 2026-05-24 | 适用于所有回测报告导出格式（Markdown、未来 PNG）

---

## 一、必须包含的声明

### 报告头部（紧接标题后）

```
⚠️ 重要提示：以下内容为历史数据回测结果，不构成投资建议。
历史表现不代表未来收益。策略参数可能过拟合，实盘结果可能显著偏离回测结果。
```

### 报告尾部（在所有数据表格之后）

已在 `packages/app/src/utils/exportReport.ts` 中实现（v0.11.0 更新版），包括：

```
## ⚠️ 风险提示

**本报告由 QuantVault 自动生成，仅供用户自我研究参考，不构成任何投资建议。**

- 历史回测不代表未来收益。策略在历史数据上的表现（包括但不限于收益率、胜率、夏普比率、最大回撤）不构成对未来表现的任何明示或暗示的保证。
- 回测假设所有交易以收盘价成交、未考虑流动性限制。实际交易成本（手续费、印花税、滑点）可能高于设置值。
- 历史回测存在幸存者偏差、前视偏差、过拟合等固有局限。
- 用户基于本报告做出的任何投资决策，风险自担。投资有风险，入市需谨慎。
```


字体大小 ≥ 报告正文字号的 70%，颜色为半透明白色或灰色，不可被裁剪。
