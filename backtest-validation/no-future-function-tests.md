## 测试 1：双均线交叉 — 基本金叉

**策略**：fast=3, slow=5, MA(close,3) 上穿 MA(close,5) → 买入，下穿 → 卖出

**手工计算**：
```
Bar 0-1: MA3 不可用（<3 bar）
Bar 2: MA3=(10.20+10.60+10.10)/3=10.30, MA5 不可用
Bar 3: MA3=(10.60+10.10+10.80)/3=10.50, MA5 不可用
Bar 4: MA3=(10.10+10.80+11.00)/3=10.63, MA5=(10.20+10.60+10.10+10.80+11.00)/5=10.54 → MA3 > MA5 首次成立
  检查 bar 3: MA3=10.50 < MA5 不可用 → 此时 MA5 刚可用
  首次 MA3>MA5 = 金叉（或至少是上穿） → Buy at bar 4 close = 11.00

Bar 5: MA3=(10.80+11.00+10.70)/3=10.83, MA5=(10.60+10.10+10.80+11.00+10.70)/5=10.64 → MA3 > MA5, 持有
Bar 6: MA3=(11.00+10.70+10.40)/3=10.70, MA5=(10.10+10.80+11.00+10.70+10.40)/5=10.60 → 持有
Bar 7: MA3=(10.70+10.40+10.85)/3=10.65, MA5=(10.80+11.00+10.70+10.40+10.85)/5=10.75 → MA3 < MA5, 死叉 → Sell at bar 7 close = 10.85
```

**预期结果**：
- Trades: 2 (1 buy + 1 sell)
- Buy @ bar 4, price=11.00
- Sell @ bar 7, price=10.85
- 收益 = (10.85 - 11.00) / 11.00 ≈ -1.4%
- 扣除手续费后略低

**状态**：⬜ 待运行验证
