# v0.12 发版闸门

> 版本 v1.0 | 2026-05-24 | 18 项检查清单

---

## 一、发版闸门原则

每项必须**亲自验证**，不能凭记忆或假设判定通过。验证方式包含截图、日志、或输出文本。

**如果任何一项不通过，不允许发版。**

---

## 二、18 项闸门检查

### G1. 构建测试

- [ ] `pnpm tauri build` 在 Windows 上通过（无 error）
- [ ] 产物文件存在：`.exe` 安装包 + `.msi` 安装包
- [ ] 文件大小合理（与上一版对比，无异常膨胀/缩水）
- [ ] 版本号在 `package.json`、`tauri.conf.json`、`Cargo.toml` 三处一致

**验证方式**：构建日志 + 产物截图

---

### G2. 安装测试

- [ ] 在干净 Windows 10/11 虚拟机上测试安装
- [ ] 安装过程无异常报错
- [ ] 安装到默认路径和自定义路径均正常
- [ ] SmartScreen 提示文案符合预期（"已保护你的电脑"→ "更多信息"→ "仍要运行"）
- [ ] 安装后在开始菜单有快捷方式

**验证方式**：虚拟机安装过程录屏或截图序列

---

### G3. 卸载测试

- [ ] 通过 Windows 设置→应用→卸载，卸载成功
- [ ] 卸载后安装目录已清除
- [ ] 卸载后开始菜单快捷方式已清除
- [ ] 注意：数据目录（`%APPDATA%/com.quantvault.app/`）不应被卸载程序自动删除（除非用户勾选）

**验证方式**：卸载前后目录对比

---

### G4. 首次启动测试

- [ ] 首次启动展示入门引导（非空白界面）
- [ ] 入门引导提供"加载演示数据"选项
- [ ] 选择"加载演示数据"后在 5 秒内看到 K 线图
- [ ] 14 天试用自动激活（标题栏显示"试用版"）
- [ ] 首次启动不崩溃、不卡死

**验证方式**：全新安装→首次启动截图

---

### G5. 演示数据测试

- [ ] 数据至少覆盖 200 个交易日

**验证方式**：K 线图截图 + 数据库 stock 表 count 截图

### G6. 通达信导入测试

- [ ] 准备一个含 50+ 只股票的测试用通达信 `vipdoc` 目录
- [ ] 导入上海+深圳数据各至少 20 只股票
- [ ] 导入过程显示进度（或至少不卡死）

- [ ] 导入的股票可在搜索栏搜索到
- [ ] 日线 K 线图正常渲染

**验证方式**：导入前后数据对比表

---

### G7. K 线渲染测试

- [ ] 日线 K 线缩放/拖拽流畅（无肉眼可见卡顿）
- [ ] 十字光标跟随鼠标移动无误
- [ ] 涨跌配色正确（红涨绿跌，中国市场默认）
- [ ] 成交量柱状图在 K 线下方显示
- [ ] 至少叠加 3 个指标（如 MACD + RSI + BOLL）后图表正常
- [ ] 2×2 网格模式：4 只不同股票各占一个画布，切换正常

**验证方式**：截图 + 交互录屏

---

### G8. 多周期切换测试

- [ ] 日线 → 周线 → 月线，每种周期 K 线正确
- [ ] 周线/月线数据为日线 resample（验证 OHLCV 聚合正确）
- [ ] 分钟线：60m/30m/15m/5m（需导入分钟线数据）
- [ ] 切换周期后指标重新计算且正确
- [ ] 2×2 模式下每个画布可独立切换周期
- [ ] 切换周期不丢失当前股票

**验证方式**：每种周期截图，手工算一笔 OHLCV 聚合

---

### G9. 策略回测测试

- [ ] 选一只股票 + 一个策略模板 + 默认参数 → 成功执行
- [ ] 至少测试 3 个不同策略模板，均成功执行
- [ ] 回测结果包含：资金曲线图 + 总收益率 + 胜率 + 最大回撤 + 交易次数
- [ ] 交易明细列表展示每笔交易的日期/价格/数量/盈亏
- [ ] 修改参数后重跑，结果有合理变化
- [ ] 回测报告有明显的"历史回测不代表未来收益"免责声明
- [ ] 手续费/滑点配置生效（如有配置入口）
- [ ] 空数据/极短数据（< 20 根 K 线）时友好提示而非崩溃

**验证方式**：每个策略的截图 + 结果数据合理性检查

### G10. 预警测试
