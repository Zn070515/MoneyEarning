# 冒烟测试清单

## 冒烟测试范围

冒烟测试覆盖**最核心流程**，确保基本可用。完整测试见发版闸门。

目标：**5-10 分钟内跑完，快速判断是否可以进入详细测试。**

---

## 核心功能序列（必做）

### S1. 安装 → 启动 → 看到 K 线（3 分钟）

- [ ] 在 Windows 10/11 全新安装
- [ ] 首次启动展示引导
- [ ] 选择"加载演示数据"
- [ ] 5 秒内看到 K 线图

**判退标准**：以上任一步失败 → STOP，修复后再测。

---

### S2. 图表交互（1 分钟）

- [ ] 滚轮缩放正常
- [ ] 拖拽平移正常
- [ ] 十字光标显示

**判退标准**：缩放/拖拽卡死或崩溃 → STOP。

---

### S3. 叠加指标（1 分钟）

- [ ] 添加 MACD → 显示正常
- [ ] 添加 RSI → 显示正常
- [ ] 指标区可拖拽调整高度

**判退标准**：MACD 数据显示明显错误（如全是 0 或 NaN）→ STOP。

---

### S4. 策略回测（2 分钟）

- [ ] 选一只演示股票
- [ ] 选"双均线交叉"策略
- [ ] 点击"运行回测"
- [ ] 回测完成，显示资金曲线 + 统计
- [ ] 报告有明显免责声明

**判退标准**：回测崩溃或结果明显错误 → STOP。

---

### S5. 预警设置（1 分钟）

- [ ] 打开预警管理
- [ ] 创建一条简单规则
- [ ] 保存成功
- [ ] 列表显示该规则

**判退标准**：创建失败或保存后丢失 → STOP。

---
