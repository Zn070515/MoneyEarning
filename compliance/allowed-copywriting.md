## 二、禁用语 → 合规替代

### 荐股类替代

| 禁止 | 允许 |
|------|------|
| 智能选股 | **条件扫描** |
| 推荐股票 | **按条件筛选** |
| 精选个股 | **条件匹配结果** |
| 潜力牛股 | **符合技术条件的股票** |
| 黑马挖掘 | **多条件交叉筛选** |
| 选股推荐 | **扫描结果列表** |

### 买卖信号类替代

| 禁止 | 允许 |
|------|------|
| 买入信号 | **条件触发标记（上）** / **做多条件触发** |
| 卖出信号 | **退出条件标记（下）** / **做空条件触发** |
| 最佳买点 | **历史回测中的条件触发位置** |
| 强烈买入 | **多项条件同时触发** |
| 黄金买点 | **条件组合触发** |

### 收益类替代

| 禁止 | 允许 |
|------|------|
| 保证收益 | **历史回测显示...**（仅陈述事实） |
| 年化30% | **在该回测区间内，年化复合增长为...**（必须标注区间） |
| 高收益策略 | **该策略在历史回测中的表现特征** |
| 稳赚 | **在特定历史期内的回测结果** |
| 赚钱 | **回测盈利**（仅限于描述历史回测结果，不可用于描述未来期望） |

### 预测类替代

| 禁止 | 允许 |
|------|------|
| 目标价15.00 | **禁止出现任何具体价格预测** |
| 明日走势 | **禁止预测未来走势** |
| 上涨概率80% | **历史回测中该信号上涨概率为...（必须标注"历史"和"回测"）** |
| 必到XX元 | **禁止出现"必到"类承诺** |
| 看涨到 | **技术分析显示形态特征为...**（仅描述形态，不预测结果） |

### 夸大类替代

| 禁止 | 允许 |
|------|------|
| 炒股神器 | **辅助分析工具** |
| 散户救星 | **面向个人投资者的数据分析工具** |
| 用了就能赚 | **帮助你更高效地做数据分析** |
| 全国第一 | **禁止绝对化排名** |
| 全覆盖 | **支持A股全市场数据**（限于客观事实） |

## 三、功能页面文案规范

### 策略回测结果区

```
✅ 允许：
- "回测区间：2020-01-01 至 2025-12-31"
- "历史回测年化复合增长：12.3%"
- "历史最大回撤：-25.8%"
- "⚠️ 历史回测不代表未来收益"
- "策略参数可能过拟合，实盘结果可能显著偏离回测结果"

❌ 禁止：
- "年化收益12.3%，非常优秀"
- "这个策略很稳"
- "推荐使用这个策略"
- "跟着这个策略做就能赚"
```

### 条件扫描结果区

```
✅ 允许：
- "技术条件匹配度：85/100"
- "符合全部3个条件的股票：12只"
- "条件1：5日均线上穿20日均线 · 条件2：成交量>20日均量1.5倍"
- "⚠️ 扫描结果仅为技术条件匹配，不构成投资建议"

❌ 禁止：
- "评分最高的3只牛股"
- "推荐买入以下股票"
- "今日最佳选股"
- "跟着买就对了"
```

### 预警管理区

```
✅ 允许：
- "价格突破预警：600519 > 1800.00"
- "均线交叉预警：5日均线上穿20日均线"
- "预警触发通知（仅供研究参考）"
- "⚠️ 预警为条件触发通知，不构成买卖建议"

❌ 禁止：
- "涨停预警"
- "大涨提醒"
- "抓住暴涨前机会"
- "最佳入场时机提醒"
```

### 策略模板列表

```
✅ 允许：
- "双均线交叉（趋势跟踪类）"
- "布林带均值回归（均值回归类）"
- "⚠️ 以下策略仅供研究参考，历史绩效不代表未来表现"

❌ 禁止：
- "高收益双均线策略"
- "稳定盈利策略"
- "跟着机构做波段"
```
