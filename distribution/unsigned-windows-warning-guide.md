# Windows 未签名应用安全提示处理指南

> 版本 v1.0 | 2026-05-24 | 面向用户的图文教程

---

## 为什么会出现安全提示？

QuantVault 目前没有购买 Microsoft 认可的代码签名证书（年费约 ¥2000-3500）。Windows 默认会拦截没有数字签名的应用，显示 SmartScreen 警告。这并不意味着软件有病毒或恶意代码，只是 Windows 无法验证发布者身份。

**这是正常的**。很多中小型独立开发者的桌面软件都会触发此提示，并非 QuantVault 独有。

---

## 如何处理（逐步骤，Windows 11 中文版）

### 第一步：下载完成后

如果 Edge 浏览器提示"此文件可能危害你的计算机"：

1. 点击下载栏右侧的 `···`（更多操作）
2. 选择 **"保留"**
3. 文件会继续下载到你的下载文件夹

### 第二步：双击安装文件时

如果看到全屏蓝色/黑色提示：

> **"Windows 已保护你的电脑"**
> Microsoft Defender SmartScreen 阻止了无法识别的应用启动。运行此应用可能会使你的电脑面临风险。

操作：
1. 点击提示文字下方的 **"更多信息"** 链接
2. 页面底部会出现新的按钮 **"仍要运行"**
3. 点击 **"仍要运行"**
4. 应用正常启动安装程序

### 第三步：安装完成后首次运行

首次运行可能再次触发 SmartScreen，重复第二步操作即可。从第二次启动开始，Windows 不会再拦截。

## 如何验证文件完整性（可选，推荐）

如果不放心，可以验证下载文件的 SHA256 校验值，确认文件未被篡改。

### 方法：PowerShell（Windows 自带）

```powershell
Get-FileHash -Path "C:\Users\你的用户名\Downloads\QuantVault_0.11.0_x64-setup.exe" -Algorithm SHA256
```

### 验证

对照片提供的 SHA256 值是否一致。每次发版会在下载页公开 SHA256 校验值。

## 会被杀毒软件误报吗？

有可能。部分国产杀毒软件（360、腾讯电脑管家）可能对未签名的安装包标记为"可疑"。这是误报（false positive），原因是没有数字签名。

### 如果被误报：

1. **360安全卫士**：在拦截弹窗中点击"更多" → "信任此程序"
2. **腾讯电脑管家**：在拦截弹窗中点击"允许"或"添加信任"
3. **火绒**：通常不会误报，如果误报可在信任区添加
4. **Windows Defender**：点击"更多信息" → "仍要运行"即可

### 为什么不上传到杀软白名单？

提交白名单需要企业资质和数字签名证书。我们正在计划中，但目前暂未完成。

## 常见问题

**Q: 没有数字签名是不是代表软件不安全？**
A: 不是。数字签名的作用是证明"这个文件确实是我发布的，没有被中间人篡改"。没有签名不代表文件有毒，只是 Windows 无法验证发布者。你可以通过 SHA256 校验值来验证文件完整性。

**Q: 会不会有病毒？**
A: QuantVault 是完全开源/源码可审计的软件。我们提供 SHA256 校验值供验证。你也可以在安装前上传文件到 [VirusTotal](https://www.virustotal.com) 自行扫描。

**Q: 什么时候会有数字签名？**
A: 计划在确认产品有足够用户需求后，用第一笔收入购买代码签名证书。

> **给用户的建议**：如果你经常使用独立开发者的小众软件，这个流程会经常遇到。关键不是有没有数字签名，而是你是否信任软件来源。SHA256 校验 + VirusTotal 扫描可以提供额外的安全感。
