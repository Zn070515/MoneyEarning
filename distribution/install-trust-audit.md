# 安装信任审计

> 版本 v1.0 | 2026-05-24 | 用户安装全链路障碍分析

---

## 一、安装全链路

```
用户看到下载链接
    ↓
点击下载 .exe / .msi
    ↓ 步骤1：浏览器安全提示
Windows SmartScreen 拦截 ⚠️ 最大阻碍
    ↓ 步骤2：用户需点击"更多信息" → "仍要运行"
杀毒软件误报/隔离 ⚠️
    ↓ 步骤3：可能需要添加白名单
Windows 管理员权限弹窗 ⚠️
    ↓ 步骤4：确认安装
安装向导（NSIS）
    ↓ 步骤5：选择安装目录
安装完成 → 首次启动
    ↓ 步骤6：SmartScreen 再次拦截（首次运行）
进入应用
```

## 三、当前状态评估

**SmartScreen 初次运行体验**：
- 无代码签名 → 每次新版本首次运行都会触发 SmartScreen
- 用户看到的提示（Windows 11 中文）："Windows 已保护你的电脑。Microsoft Defender SmartScreen 阻止了无法识别的应用启动。运行此应用可能会使你的电脑面临风险。"
- 用户需要点击"更多信息"链接 → 然后点击"仍要运行"按钮

**可立即改善的措施**：
1. ✅ 提供安装教程（见 installer-user-guide.md），带截图引导用户操作
2. ✅ 提供 SHA256 校验值，让懂技术的用户可以验证文件完整性
3. ✅ 在官网页说明"为什么会提示安全问题"，降低用户疑虑
4. ⬜ 向 Microsoft 提交应用进行信誉评估（通过 Windows Defender 门户）
5. ⬜ 向 360、腾讯电脑管家提交白名单审核
6. ❌ 购买 EV Code Signing Certificate（≈¥2000/年，长期推荐但不紧急）

## 四、签名证书购买建议（中期）

| 供应商 | 年费（约） | 优势 |
|--------|:------:|------|
| DigiCert | ¥2800-3500 | 全球认可度最高，SmartScreen 信誉累积最快 |
| Sectigo | ¥1800-2500 | 性价比高 |
| GlobalSign | ¥2000-2800 | 中规中矩 |

**流程**：购买 → 企业身份验证（1-3天）→ 签发证书 → 配置到 Tauri 构建流程

**现阶段策略**：暂不购买，先通过文档和教程正面解决安装信任问题。待 Alpha 测试验证有真实用户愿意付费后，用第一笔收入购买证书。

## 五、VirusTotal 扫描计划

每次发版前：
1. 上传 .exe 和 .msi 到 [VirusTotal](https://www.virustotal.com)
2. 检查误报引擎数量
3. 如果>3个引擎误报 → 向对应厂商提交 false positive 报告
4. 记录每次扫描结果到 release 文档

---

> **结论**：当前最大安装阻碍是 SmartScreen。短期通过文档降低用户疑虑，中期购买签名证书彻底解决。
