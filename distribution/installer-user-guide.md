## QuantVault 安装教程（Windows 10/11）

### 第一步：下载

从官网下载页面选择 `.exe` 安装包（推荐）或 `.msi` 安装包。

### 第二步：浏览器提示处理

如果 Edge/Chrome 提示 **"此文件可能危害你的计算机"**：

- Edge：点击下载栏右侧 `···` → **保留**
- Chrome：点击 **保留** 或 **继续下载**
- 360浏览器：在弹出的警告中选择 **"信任"** 或 **"保留"**

### 第三步：运行安装

双击下载的 `.exe` 文件。

如果出现 **"Windows 已保护你的电脑"** 的蓝色提示：

1. 点击 **"更多信息"**
2. 点击 **"仍要运行"**
3. 安装程序正常启动

### 第四步：安装向导

1. 选择语言：**简体中文** / English
2. 选择安装路径（默认即可，或自定义）
3. 勾选"创建桌面快捷方式"
4. 点击 **安装**
5. 等待安装完成（约 10-30 秒）

### 第五步：首次启动

1. 桌面双击 **QuantVault** 图标
2. 如再次出现 SmartScreen 提示，重复第三步操作
3. 应用启动，自动初始化数据库
4. 首次使用引导会帮你快速上手

## 文件校验（可选）

验证下载的文件是否完整、未被篡改：

1. 打开 PowerShell（Win + X → 终端）
2. 运行命令：
```
Get-FileHash "C:\Users\你的用户名\Downloads\QuantVault_0.11.0_x64-setup.exe" -Algorithm SHA256
```
3. 对比官网公布的 SHA256 值是否一致

## 如何卸载

### 方法一：Windows 设置

1. 打开 **设置** → **应用** → **已安装的应用**
2. 搜索 **QuantVault**
3. 点击 `···` → **卸载**

### 方法二：开始菜单

1. 打开 **开始菜单** → **QuantVault** 文件夹
2. 点击 **卸载 QuantVault**

### 注意

- 卸载不会自动删除你的数据文件
- 数据存储在 `C:\Users\你的用户名\AppData\Roaming\com.quantvault.app\`
- 如需完全清除，手动删除上述文件夹

---

## 如何备份数据

见软件内 **设置** → **数据管理** → **备份数据库**。

建议每 7 天备份一次。备份文件可复制到 U 盘或云盘保存。

## 遇到问题？

| 问题 | 解决 |
|------|------|
| 安装时提示缺少运行库 | 安装 [Microsoft Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe) |
| 打开后闪退 | 检查杀毒软件是否拦截，添加信任后再试 |
| 安装后桌面没有图标 | 在开始菜单搜索 "QuantVault" |
| 360提示风险 | 此为误报。在拦截弹窗中点击"更多" → "信任此程序" |
| 数据在哪里？ | `%APPDATA%\com.quantvault.app\` |
