# Alpha 测试脚本

> 版本 v1.0 | 2026-05-24 | 发给测试用户的操作指南

---

## 致测试用户

您好！感谢参与 QuantVault 的 Alpha 测试。

这是一款**本地离线的 A股技术分析桌面软件**。重要提醒：
- **不推荐股票，不构成投资建议**
- 数据完全存储在您的电脑上，不上传
- 所有分析结果仅供参考研究

下面有 6 个简单任务，完成大约需要 30-45 分钟。请按顺序操作，遇到任何问题请记录下来。

## 测试准备

### 需要的东西

- [ ] Windows 10 或 11 电脑
- [ ] 通达信软件（如果电脑有的话）
- [ ] 30-45 分钟时间

### 安装步骤

1. 从提供的链接下载 `QuantVault_Setup_0.12.0_x64.exe`
2. 双击运行安装包
3. **可能看到 Windows 提示"已保护你的电脑"**
   - 这是正常现象（未签名的软件都会有）
   - 点击"更多信息"
   - 点击"仍要运行"
4. 按照安装向导完成安装
5. 启动软件

**记录**：安装过程中遇到什么问题了吗？（特别是 SmartScreen 提示）

## 任务 1：首次见面（预计 3 分钟）

启动软件后：

1. 你看到的第一个画面是什么？
2. 有没有出现引导向导？
3. 选择"加载演示数据"
4. 等待加载完成
5. 你是否看到了 K 线图？

**记录**：
- 第一印象如何？
- 有没有感觉迷惑的地方？
- 界面看起来专业吗？

---

## 任务 2：看看 K 线图（预计 5 分钟）

1. 在左侧搜索栏输入 `600519`（贵州茅台）
2. 等待 K 线图加载
3. 尝试：
   - 用鼠标滚轮缩放图表
   - 按住鼠标左键拖拽平移
   - 移动鼠标查看十字光标
4. 在左侧"指标"标签中，点击添加 MACD
5. 再添加一个 RSI
6. 尝试拖动指标区域分隔线

**记录**：
- K 线图加载快吗？
- 缩放/拖拽流畅吗？
- 指标叠加好用吗？
- 图表配色是否能接受？

---

## 任务 3：导入自己的数据（预计 5-10 分钟）

如果电脑里有通达信：

1. 点击菜单：数据管理 → 导入 → 通达信 .day
2. 找到通达信的数据目录（通常是 `通达信安装目录/vipdoc/`）
3. 选择 `sh`（上海）或 `sz`（深圳）目录
4. 点击"开始导入"
5. 等待导入完成
6. 导入完成后搜索一只你熟悉的股票

如果没有通达信数据：
1. 点击：数据管理 → 下载行情数据
2. 输入一个股票代码
3. 选择日期范围
4. 下载

**记录**：
- 导入过程顺利吗？
- 速度怎么样？
- 导入后数据是否正确？
- 有没有报错？

---

## 任务 4：跑一次策略回测（预计 10 分钟）

1. 选择一只你熟悉的股票
2. 左侧切换到"策略"标签
3. 选择一个策略模板（如"双均线交叉"）
4. 保持默认参数，点击"运行回测"
5. 查看结果：
   - 资金曲线
   - 交易统计（总收益率、胜率、最大回撤等）
   - 交易明细列表

6. 尝试修改参数（如将快线从 5 改为 10）
7. 再次运行，对比结果

**记录**：
- 回测运行快吗？
- 结果展示清晰吗？
- 理解每个数字的含义吗？
- 有没有你觉得不合理的交易记录？
- 回测报告的免责声明看到了吗？

---

## 任务 5：设置一个条件提醒（预计 5 分钟）

1. 点击菜单：预警 → 预警管理
2. 点击"新增规则"
3. 选择一只你关心的股票
4. 设置一个简单的条件（如"价格突破 XX 元"）
5. 保存规则
6. 查看预警列表

**记录**：
- 设置过程直观吗？
- 条件类型够用吗？
- 理解每个选项的含义吗？
- 有没有看到风险提示？

---
