## 一、目标用户画像

### 理想 Alpha 测试用户

| 维度 | 描述 |
|------|------|
| 年龄 | 30-55 岁 |
| 投资经验 | 2 年以上 A 股实盘交易经验 |
| 技术能力 | 基础的电脑操作能力（会用通达信即可） |
| 分析习惯 | 日常使用通达信/同花顺/东方财富看盘 |
| 数据情况 | 电脑里已有通达信 .day 数据文件 |
| 付费意愿 | 对工具付费不排斥（买过软件/课程） |
| 判断力 | 理解"软件不能保证赚钱" |

### 排除的用户

| 类型 | 原因 |
|------|------|
| 完全没炒过股的 | 无法评估软件价值 |
| 期望"自动赚钱"的 | 会失望，会差评 |
| 只用手机端的 | QuantVault 是 Windows 桌面软件 |
| 只会打开"推荐股票"列表的 | 不理解工具定位 |
| 电脑是 Mac/Linux 的 | 暂不支持 |

## 二、用户分层

### 层级 A：核心测试用户（3-4 人）

**特征**：
- 有通达信数据积累（1 年以上）
- 有自己的交易策略逻辑
- 会做手动回测/复盘
- 对分析工具有明确诉求

**测试重点**：
- 数据导入是否顺畅
- 回测功能是否符合直觉
- 策略模板是否有用
- 发现的 Bug 是否有价值

### 层级 B：普通测试用户（3-4 人）

**特征**：
- 用通达信/同花顺看盘
- 不太做系统回测
- 主要用 K 线 + 指标

**测试重点**：
- 安装是否遇到阻碍
- 第一次使用能否看懂
- 免费版功能是否够用
- 是否会愿意付费

### 层级 C：边缘测试用户（2-3 人）

**特征**：
- 技术能力较弱
- 不太懂英文（ME Script 名称可能困惑）
- 对电脑操作不太熟练

**测试重点**：
- 安装门槛测试
- 帮助文档可读性测试
- UI 直觉性测试

## 三、招募渠道

| 渠道 | 方式 | 预期 |
|------|------|------|
| 股票交流群 | 私下邀请，说明是测试非卖货 | 2-3 人 |
| 知乎私信 | 邀请技术分析类内容下的活跃答主 | 1-2 人 |
| 朋友/熟人 | 熟人圈里炒股的 | 2-3 人 |
| 通达信论坛 | 发布测试招募（合规文字） | 1-2 人 |
| 闲鱼/淘宝 | 前期关注者 | 1-2 人 |

### 邀请话术

```
您好，我独立开发了一款本地离线的 A股技术分析桌面软件，
目前在进行 Alpha 测试阶段，希望能找几位有经验的投资者试用和反馈。

关于软件：
- 完全本地离线，不上传数据
- 兼容通达信数据导入
- 有 K线、技术指标、策略回测等功能
- 不是荐股软件，不推荐股票

测试要求：
- 日常使用通达信看盘
- 愿意花 30-60 分钟试用
- 能提供真实反馈

作为感谢：测试结束后赠送正式版激活码一枚（价值 ¥199）。

如果感兴趣，请回复。如果没空，也完全理解。
```
