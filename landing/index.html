常见问题

你可能会问

我不会编程，能用这个做回测吗？+
完全可以。QuantVault 的设计目标就是让不会编程的普通投资者也能做策略回测。20 个策略模板开箱即用——选择模板、调整参数（如均线周期、止损百分比）、选择股票和时间范围，一键运行。完整回测报告自动生成：年化收益、夏普比率、最大回撤、胜率、盈亏比、逐笔交易明细。高级用户可以用 ME Script 编写自定义策略，语法对标通达信公式。
回测结果可信吗？会不会像某些平台一样失真？+
QuantVault 的回测引擎在本地运行，使用完整的历史日线数据，包含真实的滑点模拟和手续费计算（默认单边 0.025%，可调）。不会出现同一策略在不同平台结果天差地别的问题。我们建议回测时使用至少 3-5 年数据、覆盖牛熊周期，并通过 Walk-Forward 检验策略的稳健性。如果回测收益好得不真实——那大概率确实是过拟合了。
和通达信/同花顺有什么区别？+
通达信和同花顺的核心功能是实时行情和交易下单，量化分析能力有限——通达信不支持策略回测，同花顺的回测功能非常基础。QuantVault 专注离线量化分析：策略回测、选股扫描、形态识别、筹码分析、风险度量。它是通达信/同花顺的补充而非替代。很多用户的工作流：通达信盘中看盘 → QuantVault 盘后回测复盘。
我的数据会被上传到服务器吗？+
不会。QuantVault 是 100% 本地应用——所有数据 AES-256 加密存储在您电脑上。不连接任何服务器（除了你主动发起的公开 API 数据下载）。你的自选股、交易记录、策略代码、回测结果——全部在你自己的硬盘上。你可以用 Wireshark 抓包验证。
安装时 Windows Defender 报警怎么办？+
QuantVault 暂未购买 Microsoft 代码签名证书，所以 Windows Defender 或 SmartScreen 可能弹出安全警告。这是正常现象，不是病毒。操作步骤：点击"更多信息"→"仍要运行"即可继续安装。安装后如果遇到杀软拦截，将安装目录添加为信任区即可。
