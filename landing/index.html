常见问题

你可能会问

我的自选股和交易记录会被上传到服务器吗？+
绝对不会。QuantVault 是 100% 本地应用——所有数据 AES-256 加密存储在您电脑的 SQLite 文件里。无遥测、无埋点、无后台进程、零网络依赖。核心分析功能永远离线可用。你可以用 Wireshark 抓包验证——除了你主动发起的公开 API 数据下载请求，没有任何对外连接。
换了电脑，激活码能迁移吗？+
专业版激活码绑定机器指纹（SHA-256 哈希），更换电脑时联系客服免费重新生成（同一用户免费迁移一次）。数据库文件（.qvault）可以直接复制到新电脑，所有历史数据、回测记录、复盘笔记一键迁移。
和通达信/同花顺/ThinkorSwim 的区别是什么？+
QuantVault 专注离线量化分析——策略回测、选股扫描、形态识别、筹码分析、风险度量。不提供实时行情和交易下单，是通达信/同花顺的补充而非替代。你的通达信公式可以直接粘贴到 ME Script 编辑器运行。对标 ThinkorSwim 的回测深度，但你的数据不上传任何服务器。
Mac 和 Linux 能用吗？+
主力支持 Windows 10/11 x64。macOS（Apple Silicon + Intel）和 Linux（x64）版本同步发布，核心功能完全一致。部分系统级功能（如机器指纹采集）在不同平台有细微差异。
